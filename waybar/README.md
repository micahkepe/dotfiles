# Waybar Configuration

Waybar configuration with Omarchy.

## 802.1x Configuration

We change Omarchy's default Waybar to use `nm-applet` instead of Impala by
setting `"network"` &rarr; `"on-click"` to `"nm-applet --indicator"`.
